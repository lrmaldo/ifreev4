🎯 FormResponseController - API para guardar y listar respuestas

📱 Vista responsive - Tabla adaptativa desktop/móvil
🔍 Filtros avanzados - Búsqueda por MAC y fechas

📊 Datos de Prueba

  • Respuestas con datos realistas (nombres, emails, teléfonos)

🔑 Acceso al Sistema

Para probar la funcionalidad:

  1. Ir a /login
  2. Usar credenciales: admin@example.com / password
  3. Navegar a /admin/zonas
  4. Hacer clic en "Ver Respuestas" en la zona deseada

Rutas disponibles:

  • GET /admin/zonas/{id}/form-responses - Ver respuestas (admin)
  • GET /zonas/{id}/form-responses - Ver respuestas (cliente)
  • POST /form-responses - Guardar respuesta
